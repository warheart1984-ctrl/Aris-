# ForgeChat

ForgeChat is a local-first AI shell that is designed to sit in front of your own model APIs.
Add product docs, API notes, and operating procedures to this knowledge folder so retrieval can ground answers.

Tips:
- Put specs and playbooks here.
- Use fast mode for shorter responses and lower latency.
- Point FORGE_API_URL at your own chat inference endpoint to replace the built-in mock provider.
