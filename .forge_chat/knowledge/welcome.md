

# Welcome to ARIS

ARIS (Advanced Repo Intelligence Service) is a governed intelligence system.

This is not a standard AI chat shell.

All actions are:
- reviewed
- executed
- verified

under system law.

---

## How it works

Requests follow a governed route:

Jarvis → Operator → Forge → Forge Eval → Outcome

Outcomes are recorded in:
- Hall of Fame (verified success)
- Hall of Shame (incorrect results)
- Hall of Discard (rejected actions)

---

## What you can do

- Run governed repo plans
- Inspect system behavior
- Test validation vs integrity boundaries
- Observe how decisions are made

---

## What to expect

- Some actions will be rejected
- Some will fail validation
- Some will trigger system safeguards

Failures are classified:
- validation
- correctness
- integrity
- system_fault

Only integrity failures affect system trust.

---

## First step

Run a simple governed plan and observe the system route and outcome.

---

## About the interface

This interface runs on a local-first shell (ForgeChat),
but all behavior is controlled by ARIS governance.
Only integrity failures affect system trust.
Validation and correctness failures do not alter trusted session state.
