# prototypes

This folder is the experimental surface for one-off UI or workflow prototypes.

It is where concept work can live before it is promoted into the packaged
runtime or retired as a draft.

## Owns

- experimental prototype files
- one-off concept surfaces that are not yet promoted into the main runtime

## Does Not Own

- the packaged desktop/app lanes in [`../evolving_ai/aris_demo/`](../evolving_ai/aris_demo/)
- the live source-of-truth README path in [`../README.md`](../README.md)
- release artifacts in [`../release/`](../release/)
